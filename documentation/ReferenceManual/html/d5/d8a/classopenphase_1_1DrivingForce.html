Calculates minimum number of phase field iterations for a given time step.

This function calculates the maximum time step for the phase field equation time integration.

This function uses two different stability criteria:

1) Neumann stability criterion for phase-field equation: dt < 0.5*dx^2/(InterfaceEnergy*InterfaceMobility) (TheorLimit is set to 1.0 by default); 2) Numerical criterion, where the maximum phase-field increment in the whole simulation domain has to be lower than 1E-3 (NumerLimit is set to 1E-3 by default).
