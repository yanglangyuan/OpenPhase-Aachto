Write average Plastic strain over time. 

This function will create tabulated data on the average strain in the system. Each time this function is called, a new row will be written in the specified file name for the current time step. If the file is not present it will be created, if the phase already exists, the new data will be appended!

Write total composition over a straight line in separate files. 

This function will create a separate file each time it is called, with a different file name according to the current time step. In this file, the tabulated composition data will be written down along a single line along a given axis with type = "{}\+X"{}, "{}\+Y"{} or "{}\+Z"{}. The line can be positioned with a point on it given with "{}x"{}, "{}y"{} and "{}z"{}. The output can be weight fraction with type = "{}\+WF"{}, weight percent "{}\+WP"{}, mole fraction "{}\+MF"{} and mole percent with "{}\+MP"{}.
